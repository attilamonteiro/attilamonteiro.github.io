Projetos de Ciência de Dados

Construção de um Programa de Fidelidade com Clusterização de Clientes

Utilizando dados públicos de compras de um e-commerce online, eu usei Python, Estatística e técnicas de Clusterização de Machine Learning, para encontrar grupos de clientes com características semelhantes para formar um programa de Fidelidade e direcionar ações do time de Marketing.

As ferramentas usadas nesse desafio foram:

  • Git e Github
  • Python - Pandas, Matplotlib, Scikit-learn
  • Jupyter notebooks
  • K-Means, Hierarquical Clustering, DBScan
  • AWS Cloud ( EC2, S3, Postgres, SQLite)
  • Metabase Data Visualization

Cálculo da Propensão de Compra para priorizar Clientes de Cross-Sell.

Treinei um algoritmo de Machine Learning para definir a propensão de compra de uma base de clientes, afim de fornecer uma lista ordenada de clientes pela maior propensão de compra para o time de produto fazer uma oferta.

As ferramentas usadas nesse desafio foram:

  • Git e Github
  • Python - Pandas, Matplotlib, Scikit-learn
  • Jupyter notebooks
  • Random Forest, ExtraTrees, Polinomial Regression
  • Heroku Cloud
  • API com Flask
  • Google Sheets

Identificação de imóveis para compra e revenda a fim de maximizar o lucro

Identificação de imóveis abaixo do preço médio de venda e definição do preço ideal de revenda, a partir de uma análise exploratória de dados em Python.

As ferramentas usadas nesse desafio foram:

  • Python - Pandas, Numpy e Seaborn

  • Mapas interativos com Plotly e Folium
  • Heroku Cloud
